# Система мониторинга FastAPI сервиса

Система мониторинга для FastAPI сервиса с ONNX моделью, поддерживающая настраиваемые алерты и цветное логирование.

## Структура проекта
````
step4_monitoring/
├── main.py
├── src/
│   ├── __init__.py
│   ├── monitor.py
│   ├── logger.py
│   └── config.py
├── config/
│   └── monitoring_config.yaml
├── test_images/
│   └── sample.jpg
├── requirements.txt
└── README.md
````
## Возможности

- Мониторинг endpoints `/health` и `/predict`
- Измерение времени ответа и P95 латенси
- Расчет error rate и отслеживание последовательных ошибок
- Цветные алерты (зеленый/желтый/красный)
- Конфигурируемые пороговые значения
- Структурированное логирование в JSON формате
- Периодическое тестирование инференса
- Автоматическая ротация логов
